## D/ Main Intent & User Context

### Search intent

**Dominant:** Commercial investigation — users are actively comparing loyalty vendors to build a shortlist for purchase decisions.

**Secondary intents:**
- Informational — understanding what a "loyalty platform provider" is vs. a "managed service"
- Technical/Architectural — investigating API-first, MACH-certified, headless solutions
- Navigational — accessing Gartner Peer Insights for real user ratings

### User context (ICP) = Core user problems

1. **High acquisition costs** — need to shift focus from acquisition to profitable retention
2. **Technical complexity** — managing integrations with existing CRM, POS, and ecommerce systems
3. **Data fragmentation** — unifying first-party and zero-party data into a single customer view
4. **Vendor lock-in** — avoiding rigid, expensive monolithic contracts
5. **Engagement fatigue** — moving beyond "earn-and-burn" to emotional loyalty via gamification and AI
6. **Decision overwhelm** — too many vendors, no clear framework for comparing them

### What this article MUST achieve

1. **Categorize** loyalty platform providers by segment (enterprise, mid-market, Shopify-native, headless/API-first) so readers can find their fit
2. **Provide evaluation criteria** — what features actually matter for ecommerce brands
3. **Position Joy** as the recommended Shopify-native loyalty platform with the highest flexibility
4. **Cover secondary concerns:** migration complexity, implementation timelines, pricing transparency
5. **Address "Gartner" sub-keyword** — reference Forrester Wave and Gartner Peer Insights as authority benchmarks
