## Core Feature Set

### Points Program
- Diverse earning formats (purchases, social interactions, profile completion, etc.)
- Reward points for engagement with social channels
- Flexible point expiration (FIFO — smart point expiration that expires oldest points first)

### Referral Program
- Fraud prevention built-in
- Customizable reward structure for both referrer and referee
- Trackable referral links and discount codes

### VIP Tiers
- Manual tier assignment available
- Perks system with exclusive options: early access, exclusive collections
- Flexible tier qualification logic — fully customizable criteria
- Tier demotion and grace period control

### Loyalty Onsite Touchpoints
- Multiple touchpoints that integrate seamlessly without disrupting customer experience
- Floating widget, inline widget, loyalty page, checkout integration (Shopify Plus)
- Customer account integration
- Thank You page widget
- Points calculator on product pages

### Analytics & Reporting
- Basic metrics dashboard
- Advanced reports: Assisted Orders revenue, ROI, CLV, customer retention rate
- BigQuery pipeline for advanced data analysis
- 12+ report types

### Integrations (30+)
- **Email/SMS:** Klaviyo, Omnisend, Mailchimp
- **Reviews:** Judge.me, Okendo, Loox
- **Support:** Gorgias
- **Automation:** Shopify Flow (22+ triggers, 7+ actions)
- **Wallets:** Apple Wallet, Google Wallet
- **Other:** Zapier, POS
- **Extensibility:** API, webhooks, JS SDK
