# Joy Brand Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Purple | #06048C | Primary brand color, gradient base |
| Coral Pink | #DC6578 | Accent color, gradient highlight |
| Sky Blue | #01A6EA | Secondary accent, links, highlights |
| White | #FFFFFF | Text on dark/gradient backgrounds |

## Gradient

The Joy brand gradient flows from Deep Purple (#06048C) through Coral Pink (#DC6578) to Sky Blue (#01A6EA). Used as the fixed background for all blog post images.

## Fonts

All font files are in this directory:
- DMSans-Regular.ttf
- DMSans-Medium.ttf
- DMSans-Bold.ttf
